# Vue 3 App Shell

Vue 3 + Vite SPA with Vue Router, Pinia store, and typed gateway API client.

## Scaffold

- **nodeKind:** `ui`
- **patternKind:** `spa`
- **techStack:** ['Vue', 'TypeScript', 'Vite']
- **templatesApiVersion:** `1`

## Required TemplateContext

Use manifest `contextMapping` and file path placeholders (`{{ .AgentName }}`, etc.).

## Files

- `ui/package.json` ← `package.json.tmpl`
- `ui/vite.config.ts` ← `vite.config.ts.tmpl`
- `ui/tsconfig.json` ← `tsconfig.json.tmpl`
- `ui/src/App.vue` ← `App.vue.tmpl`
- `ui/src/router/index.ts` ← `router.ts.tmpl`
- `ui/src/api/client.ts` ← `client.ts.tmpl`
- `ui/src/stores/index.ts` ← `store.ts.tmpl`
